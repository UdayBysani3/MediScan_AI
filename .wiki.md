Project Summary
MediScan is an AI-powered full-stack web application designed for medical image analysis, aimed at assisting healthcare professionals in diagnosing conditions from uploaded images. The platform enhances diagnostic accuracy and efficiency while complying with the DPDP Act 2023. Key features include a user-friendly interface, email/password authentication, and a medical guidance system that offers health recommendations based on AI analysis. MediScan focuses on improving patient outcomes by providing reliable diagnostic insights, particularly in skin disease detection, diabetic retinopathy, brain tumors, and BMI calculations. Recent enhancements include an improved navbar featuring the MediScan logo, a revamped home page with updated branding, and a redesigned dashboard with enhanced UI components and layout.

Project Module Description
The project consists of several key functional modules:

Authentication: Secure login and registration using email and password without OTP verification.
Image Upload: User interface for uploading medical images for analysis, enhanced with animations and an improved loader.
AI Analysis: Backend processing of images to generate predictions and insights using Keras models.
Dashboard: User dashboard displaying scan history and usage statistics, now with a new layout and improved visual effects.
Pricing: Subscription tiers for premium features and unlimited access, compliant with the DPDP Act 2023.
Responsive UI: Modern design components for an engaging user experience.
Model Selector: Component for selecting AI models based on user needs.
BMI Calculator: Module for calculating Body Mass Index with health recommendations.
Medical Guidance System: Provides precautions for normal results and doctor recommendations for detected diseases, including comprehensive information about doctors and medications.
Medical Services: Dedicated sections for each analysis service.
